## Installation

1. Clone the repo and move into the direction:

```sh
git clone https://github.com/libin-code/learn-threejs-with-me &&
cd learn-threejs-with-me/code/chap_7
```

2. Install packages:

```sh
npm install
```

3. Start project:

```sh
npm run dev
```
